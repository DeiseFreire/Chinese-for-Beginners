SHOPPING 

这个怎么样? Zhège zěnme yàng? How about this one? 

我不喜欢这个。 Wǒ bù xǐhuān zhège. I didn’t like this one. 

我喜欢那个。 Wǒ xǐhuān nàgè. I liked that one. 

这件怎么样? Zhè jiàn zěnme yàng? How about this one (piece of clothing)? 

件 (jiàn) is a measure word for garments and cloth. 

这件太大了 Zhè jiàn tài dà leThis one is too big. 

这件太小了 Zhè jiàn tài xiǎo le This one is too small. 

这件太红了 Zhè jiàn tài hóng le This one is too red. 

我要那件 Wǒ yào nà jiàn I want that one 

我要那件黑的 Wǒ yào nà jiàn hēi de I want that black one
